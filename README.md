# 📘 Facebook Site

## 📌 Description
A Facebook-like social media UI built using React and Tailwind CSS.  
The project simulates feed, posts, and user interface.

## 🚀 Features
- Dynamic feed layout  
- Post components  
- Responsive UI design  
- Interactive interface  

## 🛠 Tech Stack
- React  
- JavaScript (ES6+)  
- Tailwind CSS  

## ▶️ How to Run
1. Clone the repository  
2. Run:
```bash
npm install  
npm run dev
